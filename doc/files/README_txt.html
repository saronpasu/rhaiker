rhaiker - Hatena::Haiku::API ruby binding.

rhaiker.rubyforge.org/

DESCRIPTION:

WebAPI-library for Japanese micro-blog ‘Hatena::Haiku’. This library create only use Standard-Library on ruby-1.8.x.

Hatena::Haiku h.hatena.ne.jp/

Hatena::Haiku::API h.hatena.ne.jp/api

FEATURES/PROBLEMS:

  • Still does not support ‘Upload ImageFile’.

SYNOPSIS:

  see exsample/*.rb

INSTALL:

gem install rhaiker
